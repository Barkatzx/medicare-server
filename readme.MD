# Node.js + TypeScript E-Commerce Backend with Prisma & Supabase

A robust backend for e-commerce applications, built with Node.js, Express, and TypeScript, integrated with Prisma ORM and Supabase for PostgreSQL, authentication, and scalable product/image management.

---

## Features

- **User Authentication & Management**  
  Secure signup, login, and session management using Supabase Auth.

- **Product Management**  
  Add, update, delete, and list products with full type safety via Prisma.

- **Category & Inventory Management**  
  Organize products by category and track stock levels.

- **Shopping Cart & Orders**  
  Add products to cart, place orders, and manage order history.

- **Image & File Storage**  
  Upload and manage product images using Supabase Storage.

- **Admin & User Roles**  
  Role-based access control for admin operations and regular users.

- **Type-Safe Database Access**  
  Use Prisma ORM for PostgreSQL to ensure robust queries and migrations.

- **RESTful API Endpoints**  
  Clean and extendable endpoints ready for front-end integration (Flutter, React, etc.).

- **Scalable & Production-Ready**  
  Designed for smooth scaling and easy integration with Supabase services.
